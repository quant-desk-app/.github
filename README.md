# QuantDesk

High-Performance Modular Trading Protocol & Institutional Developer Suite.

QuantDesk is a next-generation decentralized trading protocol built on Solana, designed for professional traders, quantitative funds, and automated execution agents. It provides a unified, low-latency synthetic perpetual clearing layer with institutional-grade risk engine design.

## 🏛️ Ecosystem Overview

This organization hosts the public components of the QuantDesk ecosystem, enabling developers to build, backtest, and deploy automated trading strategies.

| Repository | Description | Status |
|---|---|---|
| [**`quantdesk-sdk`**](https://github.com/quant-desk-app/quantdesk-sdk) | Unified TypeScript and Rust client libraries for market-making, order routing, and account management. | Public Beta |
| [**`docs`**](https://github.com/quant-desk-app/docs) | Developer documentation, API references, perp listing policies, and protocol specs. | Live |

## 🔒 Security & Verification

QuantDesk operates under a **Zero-Trust Security Model** for all public code interfaces:
- **No Private Keys in Source**: All SDK clients leverage standard wallet provider interfaces or secure client-side signing.
- **Audit Trails**: Smart contract interfaces are verified on-chain. Public specifications are mirrored directly from our core protocol architecture.
- **Vulnerability Reporting**: To report a vulnerability, please consult our `SECURITY.md` guidelines or open an issue under the respective repository.
